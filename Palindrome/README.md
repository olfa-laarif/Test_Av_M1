Sentence palindromes

Un palindrome est une chaîne de caractères qui peut être lue de la même façon de gauche à droite et de droite à gauche

Règles

La fonction reçoit en entrée une chaîne de caractères (input).

Elle doit renvoyer un tableau (string[]) contenant tous les palindromes détectés dans cette chaîne.
